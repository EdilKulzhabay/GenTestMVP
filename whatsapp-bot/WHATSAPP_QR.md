# QR-код для аутентификации WhatsApp на сервере

На сервере (pm2, headless) QR-код не выводится в консоль. Используйте один из способов ниже.

---

## Способ 1: HTTP-страница (рекомендуется)

1. Запустите whatsapp-bot (через pm2 или вручную).
2. Подождите 10–20 секунд, пока сгенерируется QR.
3. Откройте в браузере:
   ```
   http://ВАШ_СЕРВЕР:5112/qr-page
   ```
   Например: `http://46.247.40.26:5112/qr-page`

4. Страница сама обновится, когда QR будет готов. Отсканируйте его в WhatsApp:
   - WhatsApp на телефоне → Настройки → Связанные устройства → Привязать устройство

---

## Способ 2: Прямая ссылка на изображение

```
http://ВАШ_СЕРВЕР:5112/qr
```

Отдаёт PNG-картинку. Если QR ещё не готов — будет JSON с сообщением. Обновите страницу через 10–20 сек.

---

## Способ 3: Файл на диске

QR сохраняется в:
```
<рабочая_папка_whatsapp-bot>/.wwebjs_auth/qr.png
```

Скачайте файл на свой компьютер (SCP, SFTP) и откройте:
```bash
scp user@server:/path/to/whatsapp-bot/.wwebjs_auth/qr.png ./
```

---

## Важно

1. **Порт 5112** должен быть открыт в файрволе, если вы открываете QR с другого устройства.
2. **Один раз** отсканировав QR, сессия сохраняется в `.wwebjs_auth/`. Повторная аутентификация не нужна.
3. Если Chrome/Chromium не найден — установите:
   - Ubuntu/Debian: `sudo apt install chromium-browser`
   - Или укажите путь в `.env`: `CHROME_PATH=/usr/bin/chromium`
